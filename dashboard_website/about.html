This is a website to show the transmission of data from a LoRa node to a LoRa gateway and onto a Website. A Dragino LoRa arduino shield is attached to an arduino uno which broadcasts its payload which is then picked up by the nearest gateways. The gateways are accessible online through The Things Network's console platform. The data is sent via a nodejs application from The Things Network's console to Google's Firebase and then displayed on this website which is hosted on Firebase. The website is using Bootstrap and various JavaScript libraries.
